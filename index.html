Franklin AI Middleware

The Next Evolution of AI

Franklin is a hybrid AI engine that blends local tools with cloud intelligence, giving you:

  • Access to files and folders
  • System-aware memory and summarization
  • Log and document understanding
  • Notifications and proactive updates
  • Fallback to local TinyLlama if OpenAI is unavailable
  • FastAPI backend with Web UI
